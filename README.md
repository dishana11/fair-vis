# ERNIE Fairface Model

This repository contains a fine-tuned ERNIE model for generating neutral and factual descriptions of people in images. The model was trained on the FairFace dataset.

## Usage

To use the model, you will need to have Python and the following libraries installed:

* transformers
* torch
* accelerate
* datasets
* Pillow

You can then load the model and use it to generate descriptions of images as shown in the notebook.
